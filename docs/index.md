## Quick Start

1. Install Draco from Obsidian Community Plugins (or manually for this DIY version)
2. Add an API key in Settings → Draco → Basic → API Keys
3. Open the chat panel with the robot icon in the left ribbon
4. Start chatting!

For a full walkthrough, see [Getting Started](getting-started.md).

## What Makes Draco Different

Draco is a DIY/community version focused on:

- **No commercial dependencies** - Core features work without subscriptions
- **Bring your own model** - Use any OpenAI-compatible provider
- **Privacy-focused** - Your data stays in your control
- **Hackable** - Open source, customize to your needs
